# walkingkooka-tree-expression-function-stat
A collection of ExpressionFunction(s) that operate mostly on lists and are statistical in nature.


## Available functions

- average(number...)
- count(object...)
- max(number...)
- min(number...)
- sum(number...)

Many more functions are outstanding and remain TODO.
